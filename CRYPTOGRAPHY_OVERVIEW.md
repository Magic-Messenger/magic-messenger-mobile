# Cryptography Overview (Non-Code)

## End-to-End Encryption

Messages are encrypted on sender devices and decrypted only by recipients.

## Key Model

- Long-term identity keys
- Ephemeral session keys

## Forward Secrecy

Past messages remain secure even if keys are compromised.

## Group Messaging

Group keys are rotated when members join or leave.
